== Description ==



Download Media Library Plugin helps admins to backup the media files available in the media library of the wordpress site.



== Installation ==



1. Upload EPT Empty Plugin Template to the /wp-content/plugins/ directory 

2. Activate the plugin through the 'Plugins' menu in WordPress
